# E-Commerce Sales Analysis using SQL

## Project Overview

This project analyzes an e-commerce dataset using PostgreSQL to extract business insights related to customer behavior, product performance and revenue trends.

## Tools Used

* PostgreSQL
* SQL

## Dataset Tables

* customers
* products
* orders
* order_items
* payments

## Key Analysis Performed

* Total revenue and order analysis
* Best selling products
* Top customers
* Sales by category
* Monthly revenue trends
* Customer segmentation
* Running total revenue

## Key Insights

* Electronics category generates the highest revenue
* A small group of customers contributes a large portion of sales
* Certain products consistently outperform others
* Revenue shows steady growth over time

## Skills Demonstrated

* SQL Joins
* Aggregations
* Window Functions
* CTE & Subqueries
* Business Analysis
